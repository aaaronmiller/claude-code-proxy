🎓 Who Should Use This Fork?

✅ USE THE FORK if you:

  • Spend >$50/month on Claude/OpenAI API (compression pays for itself in ~1 month)
  • Need multi-model conversations (crosstalk feature)
  • Want usage analytics + quotas (billing, team cost tracking)
  • Require production reliability (circuit breakers prevent cascade failures)
  • Prefer GUI dashboard over CLI-only management
  • Run GPU locally (CUDA/ROCm acceleration for compression)
  • Need free tier optimization (SCOUT tier with ranked free models)

❌ STICK WITH UPSTREAM if you:

  • Want minimal single-file proxy (300 LOC, zero deps)
  • Don't need compression (already on free tier or low usage)
  • Cannot accept test coverage gap (fork has almost none)
  • Need easy mergeability with upstream (impossible now — different architectures)
  • Prefer zero-maintenance (fork has manual steps, no auto-updates yet)
